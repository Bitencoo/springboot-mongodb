# Spring Boot - MongoDB

This is an API made with SpringBoot, MongoDB and SpringFox. The goal of this API is to apply concepts as:

- Restful APIs;
- Spring Boot API;
- MongoRepository;
- MongoTemplate;
- Aggregation;
- Custom Queries.
    

## Tech Stack

**Server:** Spring Boot 2.7.8, SpringFox, Java 19

**Database:** MongoDB
